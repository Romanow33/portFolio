Countries Proyect



Mi primer proyecto individual desarrollado con ReactJS

Completamente funcional, esta aplicacion consume los datos de una API para alojarlos en una base de datos postgresSql. Utilizando herramientas FullStack como, Redux , Express, NodeJs, JavaScript, y un diseño hecho puramente en Css, esta aplicacion de turismo nos permite saber datos de paises y ver sus banderas, siendo posible filtrarlo por estos datos. Ademas podemos utilizar la herramienta de creacion para aplicar nuevas actividades para un pais en particular. ahora si podemos buscar una actividad que nos guste ver que pais la tiene y armar las balijas! si te interesa esta app podes ver su codigo AQUI!
